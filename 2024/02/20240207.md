# プログラミング
- 英論を日本語に訳す際に改行されている箇所が翻訳の邪魔をするため、改行をなくす処理をする簡単なプログラムを記述  
- Python言語使用

```python
f = open('english.txt','r+', encoding='UTF-8')

#データの取得
data = f.read()
#改行を半スペースで置換
replace_data = data.replace('\n',' ')

#ファイル書き込み
f.write('\n')
f.write('\n')
f.write(replace_data)

f.close()
```
